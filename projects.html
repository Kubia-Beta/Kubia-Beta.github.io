Router Security Analysis

I analyzed vulnerabilities in router firmware manufactured by Micronet-Union Technologies, Inc. The identified weaknesses were mapped to the Common Weakness Enumeration (CWE) database, and I provided low-cost solutions to the manufacturer. This project required expertise in HTML, JavaScript, C, C++, and ASP, along with networking and security standards.
