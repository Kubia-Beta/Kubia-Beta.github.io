Router Security Analysis

I analyzed vulnerabilities in router firmware manufactured by Huizhou TCL Communication Electronics Co. The identified weaknesses were mapped to the Common Weakness Enumeration (CWE) database, and I provided low-cost solutions to the manufacturer. This project required expertise in HTML, JavaScript, C, and ASP, along with networking and security standards.

Highlights:

  • Discovered firmware exploits affecting network security.
  • Collaborated with Micronet-Union Technologies to enhance router firmware security.
  • Provided detailed test scripts and remediation paths.
